## 1. 왜 그림으로 볼까?

다음 두 가지를 비교해 봅시다. 같은 데이터입니다.

**(A) 표로 보기**
```
펭귄 종      평균 몸무게(g)
Adelie       3700
Chinstrap    3733
Gentoo       5076
```

**(B) 막대그래프로 보기**
```
Adelie     ███████████          3700
Chinstrap  ███████████          3733
Gentoo     ████████████████     5076
```

(B)를 보는 순간 *"Gentoo가 확 무겁네!"* 가 0.5초 만에 들어옵니다. 표(A)는 숫자를 *읽어야* 알지만, 그림(B)은 *보면* 압니다.

> **시각화 = 데이터를 '읽는 것'에서 '보는 것'으로 바꾸는 일.**
> 사람의 눈은 숫자보다 *길이·위치·색*의 차이를 훨씬 빨리 알아챕니다. 그래서 분석의 마지막엔 거의 항상 그림이 등장합니다.

## 2. 차트 4종 — 언제 무엇을 쓰나

초급 단계에서는 이 네 가지면 대부분 해결됩니다. **"무엇이 궁금한가"** 에 따라 고릅니다.

| 차트 | 언제 쓰나 | 한 줄 질문 | seaborn |
|---|---|---|---|
| **막대그래프** | 그룹끼리 **비교** | "어느 쪽이 더 크지?" | `sns.barplot` |
| **히스토그램** | 한 변수의 **분포** | "값이 어디에 몰려 있지?" | `sns.histplot` |
| **산점도** | 두 변수의 **관계** | "이게 늘면 저것도 늘까?" | `sns.scatterplot` |
| **박스플롯** | 그룹별 **분포 비교** | "그룹마다 퍼진 모양이 다를까?" | `sns.boxplot` |

### 막대그래프(bar) — 비교
범주(종류)별 수치를 막대 길이로 비교합니다. "나라별 행복 점수", "요일별 매출"처럼 *"어느 것이 큰가"* 를 볼 때.

### 히스토그램(histogram) — 분포
하나의 숫자 변수가 *어떤 값에 얼마나 몰려 있는지* 봅니다. "학생들 키가 보통 어디쯤에 모여 있나?" 막대그래프와 헷갈리기 쉬운데, 히스토그램은 **하나의 변수**를 구간으로 쪼개 *개수*를 셉니다.

### 산점도(scatter) — 관계
점 하나가 관측치 하나. 두 변수를 가로·세로축에 놓고 *관계*를 봅니다. "공부 시간이 늘면 성적도 오를까?" 점들이 **오른쪽 위로 흐르면 양의 관계**입니다. 색(`hue`)으로 그룹을 나누면 더 많은 걸 볼 수 있어요.

### 박스플롯(box) — 그룹별 분포 비교
그룹마다 값이 *얼마나, 어떻게 퍼져 있는지*를 상자로 요약합니다. 상자는 가운데 50%, 가운데 선은 중앙값. "종별로 몸무게 분포가 어떻게 다른가"를 한 번에 비교.

> 💡 **고르는 법 요약**: 비교→막대, 분포→히스토그램, 관계→산점도, 그룹분포→박스.

## 3. seaborn으로 한 줄이면 끝

**seaborn**은 예쁜 차트를 *한 줄*로 그려주는 도구입니다. 형식이 거의 똑같아서 외우기 쉽습니다.

```python
import seaborn as sns
import matplotlib.pyplot as plt

sns.scatterplot(data=df, x="bill_length_mm", y="bill_depth_mm", hue="species")
plt.show()
```

공통 형식:
```
sns.차트종류(data=데이터, x="가로축 열", y="세로축 열", hue="색으로 나눌 열")
```
- `data=` 어떤 표를 쓸지
- `x=`, `y=` 어떤 열을 축에 놓을지
- `hue=` (선택) 어떤 열로 *색*을 나눌지
- 마지막에 `plt.show()`로 그림을 띄웁니다.

## 4. 한글 폰트 깨짐 — 네모(□□□) 대처법

차트 제목에 한글을 넣으면 `□□□`처럼 네모로 깨질 때가 있습니다. 컴퓨터에 한글 폰트 설정이 안 돼 있어서예요. 아래 코드로 폰트를 지정합니다.

```python
import matplotlib.pyplot as plt
from matplotlib import font_manager as fm
import os

# 폰트를 못 찾아도 '에러 없이' 넘어가게 try/except로 감쌉니다
try:
    path = "../data/BMDOHYEON.ttf"          # 이 강의에 동봉한 폰트
    fm.fontManager.addfont(path)
    plt.rcParams["font.family"] = fm.FontProperties(fname=path).get_name()
    plt.rcParams["axes.unicode_minus"] = False   # 마이너스 부호 깨짐 방지
except Exception:
    print("한글 폰트를 못 찾았어요(영문은 정상). 차트는 그대로 그려집니다.")
```

> ⚠️ **중요**: 위처럼 `try/except`로 감싸면, **폰트가 없어도 차트 셀이 죽지 않습니다.** 한글이 네모로 보일 뿐, 영문 차트는 멀쩡히 그려져요. (구글 코랩에는 기본 한글 폰트가 없어서 이 상황이 실제로 자주 생깁니다. 그래서 실습 차트는 *영문 라벨* 위주로 만들었습니다.)
