## 1. 분석은 결국 '하나의 흐름'이다

지금까지 배운 걸 따로따로 기억하면 금방 잊습니다. 대신 **하나의 흐름**으로 묶으세요. 모든 데이터 분석은 거의 항상 이 순서를 밟습니다.

```
 ① 질문        ② 불러오기      ③ 정제         ④ 그룹핑·계산     ⑤ 시각화       ⑥ 해석
 무엇을      →  데이터를    →  지저분한 걸  →  나눠서 비교   →  그림으로   →  그래서
 알고 싶나?     가져온다       깨끗이          요약한다         보여준다       뭐?
   (1장)        (2장)         (3장)           (3장)           (4장)        (1·5장)
```

> 📌 핵심: **분석은 ②~⑤(도구·기술)로 끝나지 않습니다.** 시작(①질문)과 끝(⑥해석)은 *사람의 몫*입니다. 가운데 기술은 AI가 점점 더 도와주지만, 좋은 질문과 해석은 여전히 여러분이 합니다.

## 2. 미니 프로젝트 — "타이타닉, 누가 살아남았나?"

이번 프로젝트의 질문은 이것입니다.

> **"타이타닉호에서 어떤 사람이 더 많이 살아남았을까? 그리고 그건 왜일까?"**

이 질문 하나를 데이터로 답해봅니다. 흐름은 이미 다 배운 것들입니다.

| 단계 | 할 일 | 쓰는 기술(배운 장) |
|---|---|---|
| ② 불러오기 | 타이타닉 데이터 로드 + 첫인상 | `load_dataset`, `head`, `shape` (2장) |
| ③ 정제 | 나이(`age`) 결측치 채우기 | `fillna` (3장) |
| ④ 그룹핑 | 등급별·성별 생존율 계산 | `groupby` (3장) |
| ⑤ 시각화 | 생존율을 막대그래프로 | `sns.barplot` (4장) |
| ⑥ 해석 | "그래서 무엇을 알았나" 한 문단 | 사람의 몫 (1장) |

> 🔵 **수업에서는** 강사가 ②와 ④의 *흐름*만 같이 잡습니다(핵심 셀 몇 개).
> 🟡 **나머지는 집에서** 직접 채워보세요. 정답(`_solution`)이 전부 들어 있으니, 막히면 대조하면 됩니다. **이 프로젝트를 집에서 한 번 더 완주하는 것** — 그게 이 수업의 진짜 목표입니다.

### 해석은 이렇게 (예시)
숫자를 구한 뒤엔 *말로* 풀어내야 합니다. 예:

> "1등석 승객의 생존율(약 63%)이 3등석(약 24%)보다 훨씬 높았다. 또 여성의 생존율(약 74%)이 남성(약 19%)보다 압도적으로 높았다. 이는 구조 당시 *'여성과 어린이 먼저, 그리고 1등석 우선'* 이라는 상황이 데이터에 그대로 남은 것으로 보인다."

→ 이렇게 **숫자 + 그게 무슨 의미인지**를 함께 적는 게 '해석'입니다.

## 3. 셀프스터디 로드맵 — 이제 어디로?

이 5시간은 **출발선**입니다. 여기서 멈추지 말고, 아래 순서로 한 걸음씩 더 나아가 보세요. 욕심내지 말고 위에서부터 하나씩.

### 🟢 바로 다음 (이 강의의 자연스러운 연장)
1. **판다스 더 익히기** — `merge`(표 합치기), 피벗, 더 다양한 `groupby`. (2~3장 복습 + 확장)
2. **시각화 더 그려보기** — [seaborn 갤러리](https://seaborn.pydata.org/examples/)의 예제를 따라 그리기.
3. **통계 기초** — 평균·분산·상관계수, "이 차이가 우연일까?"(간단한 가설검정).

### 🟡 그다음 (분석가의 기본 무기)
4. **SQL** — 데이터베이스에서 원하는 데이터를 직접 꺼내오는 언어. 실무 필수.
5. **머신러닝 입문** — `scikit-learn`으로 예측·분류 맛보기. (1장의 '3단계'를 직접)
6. **실전 데이터로 연습** — [Kaggle](https://www.kaggle.com)의 입문 대회(타이타닉 생존 예측!)로 손을 풀기.

### 🔴 더 멀리 (관심 분야로 깊이)
7. **AI 도구를 분석에 활용** — ChatGPT 등에게 분석을 *거들게* 하기. 단, 질문과 해석은 **내가** 한다.
8. **분야별 심화** — 예를 들어 강사가 다루는 **금융·핀테크 분석**:
   - **시계열 예측** — 비트코인·주가·금값의 *미래 흐름* 다루기 (1장에서 본 그 데이터들).
   - **이상거래 탐지** — 수상한 금융 거래를 자동으로 찾아내기.
   - → 이건 *한참 뒤*의 이야기지만, "초급을 넘어서면 이런 세계가 있다"는 걸 알아두세요.

```
 [지금 여기]
   기본기  ──→  통계·SQL  ──→  머신러닝  ──→  분야 심화(금융 시계열·이상탐지 등)
   (이 강의)                                      (강사 fintech 영역)
```

## 4. 마지막으로 — 1장으로 돌아오며

우리는 1장에서 이렇게 시작했습니다: 도구는 엑셀에서 머신러닝, ChatGPT, 에이전트로 점점 똑똑해진다고요. 그리고 한 가지를 약속처럼 깔아두었습니다.

이제 4시간의 기본기를 거쳐 그 문장을 다시 봅니다. 이제는 *느낌*이 다를 겁니다.

> ## "도구는 변해도, 좋은 질문을 던지고 결과를 해석하는 힘은 사람의 몫이다."

- 여러분은 이제 데이터를 **불러오고, 정제하고, 그룹으로 나누고, 그림으로 보고, 해석**할 수 있습니다.
- 이 기본기가 있어야, 앞으로 나올 어떤 똑똑한 AI 도구가 와도 **부려 쓰는 사람**이 됩니다.
- AI가 내놓은 답이 *맞는지 틀리는지* 알아채고, *더 좋은 질문*을 던질 수 있는 사람.

도구는 계속 바뀔 겁니다. 하지만 오늘 여러분이 잡은 *사고방식*은 바뀌지 않습니다. **그게 이 5시간의 진짜 선물입니다.**

수고하셨습니다. 이제 집에 가서, 정답을 펼쳐놓고 한 번 더 직접 해보세요. 🚀

## 📎 5장 요약 한 장

- 분석은 **하나의 흐름**: 질문 → 불러오기 → 정제 → 그룹핑 → 시각화 → 해석. 시작과 끝은 사람의 몫.
- 미니 프로젝트로 그 흐름을 *한 데이터(타이타닉)* 로 관통해봤다.
- 로드맵: 판다스·시각화·통계 → SQL·머신러닝·캐글 → AI 활용·분야 심화(금융 등).
- **"도구는 변해도, 질문과 해석은 사람의 몫"** — 이 사고방식이 이 과정의 핵심.
