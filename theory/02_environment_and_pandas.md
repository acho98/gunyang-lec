## 2. 표 데이터는 어떻게 생겼나 — 행과 열

데이터 분석에서 가장 많이 만나는 형태는 **표(table)** 입니다. 엑셀 시트를 떠올리면 됩니다.

```
        열(column) = 변수(무엇을 재는가)
        ┌─────────┬──────┬──────┐
        │  이름    │ 나이  │ 성별  │
        ├─────────┼──────┼──────┤
행(row) │  김철수  │  21  │  남   │   ← 관측치 1명
= 관측치 │  이영희  │  23  │  여   │   ← 관측치 1명
        │  박민수  │  20  │  남   │
        └─────────┴──────┴──────┘
```

- **행(row)** = 하나의 *관측치*. 위 예에서는 '사람 한 명'.
- **열(column)** = 하나의 *변수*. '나이', '성별'처럼 무엇을 재는지.

판다스에서는 이런 표를 **데이터프레임(DataFrame)** 이라고 부릅니다. 앞으로 데이터 = 데이터프레임 = '표'라고 생각하면 됩니다.

> 📌 **핵심**: "행은 관측치, 열은 변수." 데이터를 보면 항상 *"한 행이 무엇 하나를 나타내지?"* 를 먼저 물어보세요. 타이타닉 데이터라면 "한 행 = 승객 한 명"입니다.

## 3. 데이터 첫인상 읽기 — '5종 세트'

새로운 데이터를 받으면, 분석가는 항상 똑같은 **'첫인상 점검'** 을 합니다. 마치 새 사람을 만나면 이름·나이부터 묻듯이요. 다섯 가지 도구면 충분합니다.

| 명령 | 무엇을 알려주나 | 비유 |
|---|---|---|
| `df.head()` | 맨 위 몇 줄 미리보기 | "얼굴 한 번 보기" |
| `df.shape` | 행·열 개수 | "몇 명이고 몇 가지를 쟀나" |
| `df.info()` | 각 열의 타입·결측 여부 | "신상명세서" |
| `df.describe()` | 숫자 열의 요약(평균·최댓값 등) | "건강검진 요약" |
| `df.columns` | 열 이름 목록 | "목차" |

하나씩 봅시다. (예: 타이타닉 데이터 `df`)

### `df.head()` — 일단 눈으로 보기
```python
df.head()       # 위에서 5줄 (기본값)
df.head(3)      # 위에서 3줄
```
표가 실제로 어떻게 생겼는지 *눈으로* 확인합니다. 분석의 첫걸음은 언제나 "일단 보기"입니다.

### `df.shape` — 크기 파악
```python
df.shape        # 예: (891, 15)  → 891행, 15열
```
괄호 안 첫 숫자가 **행(관측치) 수**, 둘째가 **열(변수) 수**입니다. "타이타닉엔 승객 891명, 정보 15가지가 있구나."

### `df.info()` — 신상명세서
```python
df.info()
```
각 열의 **이름, 데이터 타입, 비어있지 않은 값의 개수**를 한 번에 보여줍니다.
- 여기서 **결측치(비어있는 값)** 가 있는 열을 발견할 수 있습니다. (예: 나이 열에 빈 칸이 많네?)
- 결측치 처리는 **3장**에서 본격적으로 배웁니다. 지금은 "찾아내는" 것까지.

### `df.describe()` — 숫자 요약
```python
df.describe()
```
숫자로 된 열들의 **개수·평균·최소·최대·중앙값** 등을 요약해 줍니다. "승객 평균 나이는 30세, 가장 어린 사람은 0.4세(아기!)구나."

### `df.columns` — 열 이름 목록
```python
df.columns
```
어떤 변수들이 있는지 *목차*를 봅니다. 분석할 거리를 고를 때 제일 먼저 보죠.

## 4. 열 하나만 꺼내 보기

표 전체가 아니라 **특정 열 하나**만 보고 싶을 때가 많습니다.

```python
df["age"]                  # '나이' 열만 꺼내기
df["age"].mean()           # 나이의 평균
df["sex"].value_counts()   # 성별 종류별 개수 세기
```

- `df["열이름"]` : 그 열만 쏙 꺼냅니다.
- `.value_counts()` : 어떤 값이 각각 몇 번 나오는지 세줍니다. (예: 남 577명, 여 314명) — **3장·4장에서 단골로** 씁니다.

> 💡 대괄호 안 **열 이름은 따옴표**로 감쌉니다: `df["age"]`. 따옴표를 빠뜨리는 게 초보자의 단골 실수예요.
